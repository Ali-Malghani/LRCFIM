# LRCFIM
LRCFIM (Lightweight Real-Time Cloud-Integrated File Integrity Monitoring)
is a lightweight event-driven File Integrity Monitoring (FIM) prototype
supporting real-time monitoring, SHA-256 integrity verification,
dynamic risk scoring, and cloud-integrated forensic logging.
This repository accompanies the research paper:
"LRCFIM: A Lightweight Real-Time Cloud-Integrated File Integrity
Monitoring System with Dynamic Risk Scoring and Automated Compliance"
## Features
- Real-time file-system event monitoring
- SHA-256 integrity verification
- Dynamic multi-factor risk scoring
- Event-driven architecture
- Lightweight local deployment
- Benchmark and evaluation scripts
- Sample configuration for reproducibility
